# Bibliothèque de marque

Les fichiers officiels fournis par Clément le 13 juillet 2026 sont conservés dans `public/brand` avec des noms web stables, sans espace ni accent.

## Usages actifs

| Usage | Fichier |
|---|---|
| Logo principal | `/brand/logo-officiel.png` |
| Logo vectoriel | `/brand/logo-officiel.svg` |
| Logo sur fond sombre | `/brand/logo-noir.png` |
| Favicon et icône PWA | `/brand/favicon.png` et `/brand/favicon.svg` |
| OpenGraph et partage social | `/brand/couverture-facebook.jpg` |
| Carte membre Couple | `/brand/carte-membre-couple-2026.png` |
| Carte membre Famille | `/brand/carte-membre-famille-2026.png` |

Les chemins applicatifs sont centralisés dans `src/config/brand-assets.ts`. Les variantes carte, plaque et stickers sont archivées dans le même dossier pour les futurs écrans produit et supports partenaires.

## Règles

- Utiliser le logo officiel pour l'en-tête et les communications principales.
- Préférer le PNG optimisable dans l'interface ; garder le SVG pour les supports nécessitant une mise à l'échelle.
- Ne pas renommer directement les fichiers publiés : modifier d'abord la configuration centralisée.
- Ne pas afficher une carte Couple ou Famille comme offre commerciale avant validation de son prix et de ses conditions.
